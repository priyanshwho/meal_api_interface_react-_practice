# Meals Explorer

Meals Explorer is a visually experimental recipe discovery app built around the FreeAPI meals catalog. It presents a curated meal browsing experience with editorial card layouts, dreamy pastel-red color layers, soft glassmorphism, and smooth motion so the interface feels more like a modern food magazine than a standard recipe grid.

The project is designed for exploring meals quickly and beautifully. It highlights search, category filtering, favorites, and detailed recipe viewing while keeping the overall mood soft, premium, and slightly surreal.

The visual direction leans into dusty rose, muted coral, terracotta, beige-red gradients, and blurred layered surfaces to create a memorable browsing experience with a gentle artsy edge.
